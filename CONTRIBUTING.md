# Contributing

Thanks for contributing to `ohmyvoice`.

## Development Setup

1. Install dependencies:

```bash
npm install
```

2. Run checks:

```bash
npm run format:check
npm run lint
npm run typecheck
npm run test
npm run build
```

## Pull Request Guidelines

- Keep changes focused and small.
- Update docs when behavior or configuration changes.
- Add or update tests for behavior changes.
- Include a short "what changed" and "how it was verified" in PR description.

## Commit Style

- Use clear, imperative commit messages.
- Avoid mixing refactor and feature changes in one commit.

## Reporting Bugs

When opening an issue, include:

- Expected behavior
- Actual behavior
- Steps to reproduce
- Relevant logs or command output
